MANUAL FOR BIOBOX CONTAINING PROJECT PARTIS
#7-24-15
==============================
1. Git clone this repository: https://github.com/matsengrp/bioboxpartis
2. Navigate into the input_data directory and run YamlFileGenerator.py. Here you will enter all the parameters for generating the input. Make sure the resulting file (biobox.yml) is in the input_data directory. 
3. Navigate back to the main directory and build the docker image from the docker file "docker build -t <name of image> ."
4. Now you will actually run the program using this docker command: docker run --volume="$(pwd)/input_data:/bbx/input:ro" --volume="$(pwd)/output_data:/bbx/output:rw" <name of image> <task>

You should now see the output files in the directory you specified. 
==============================
Caution: 
-Make sure the task you specified in the docker run command is the same as the one in the input biobox yaml file. 
-You must have docker installed to run this program. 
==============================
Docker run command explained:
docker run --volume="$(pwd)/input_data:/bbx/input:ro" --volume="$(pwd)/output_data:/bbx/output:rw" <name of image> <task>
-the --volume tags will link the directory on your local machine to directories within the docker container
-task is the specified task you want to run; to see all the possible executable tasks take a look at the taskfile
==============================
